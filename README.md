# Dakkar Car Rental - Gestió de Flota

Projecte de gestió de flota per a una empresa de lloguer de cotxes a Curaçao. Integra tecnologies IT per a la gestió de reserves i vehicles.

## 🚀 Instruccions d'instal·lació
1. Clonar el repositori.
2. Crear entorn virtual: `python3 -m venv venv`
3. Activar entorn: `source venv/bin/activate`
4. Instal·lar Django: `pip install django`
5. Aplicar migracions: `python manage.py migrate`
6. Executar servidor: `python manage.py runserver`
